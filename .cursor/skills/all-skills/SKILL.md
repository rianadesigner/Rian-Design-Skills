# All Skills — 设计技能总路由（@all-skills）

## 第一步（必须）

打开并遵循仓库内完整路由文档（含总览表、平台列、路径、组合推荐）：

**`All skills.md/All skills.md`**（相对本仓库根目录）

若该文件已在上下文中，则直接按其「快速决策流程」执行。

## 第二步

从上述文档中选定 **最少必要** 的 `SKILL.md`，路径形如：

- `All skills.md/14.ui-ux-design/SKILL.md`
- `All skills.md/2.impeccable/.agents/skills/audit/SKILL.md`
- `All skills.md/1.ui-ux-pro-max-skill/.claude/ui-ux-pro-max/SKILL.md`

**不要**在无差别情况下并行载入 23 个子包全文。

## 第三步

执行具体子技能指令；需要查组件库源码时，再查阅 `opensource-ui/vendor-repos/`（见路由文档 § 七）。
